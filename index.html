Мои ключевые проекты

twitch-ai-chatbot
Node.js AI Integration Twitch API

Интеллектуальный чат-бот для стриминговой платформы Twitch, работающий на базе искусственного интеллекта. Он способен анализировать сообщения зрителей и генерировать живые, контекстные ответы.

В чём польза проекта:

Помогает стримерам автоматизировать модерацию, развлекать и удерживать аудиторию интерактивом, отвечать на частые вопросы зрителей в режиме реального времени, даже когда автор увлечен игровым процессом.

BeamNG.drive-map-generator-v3
Python / C# Procedural Gen BeamNG Mods

Продвинутый инструмент для процедурной и алгоритмической генерации карт, дорог и ландшафтов для физического автосимулятора BeamNG.drive.

В чём польза проекта:

Избавляет от необходимости вручную создавать ландшафты в тяжелых редакторах. Позволяет игрокам в пару кликов генерировать абсолютно новые, уникальные трассы для проведения тестов физики мягких тел и заездов.

IR-remote-mobile
Mobile App IR Blaster API C++ / Java

Мобильное приложение, превращающее смартфон со встроенным инфракрасным портом (IR Blaster) в универсальный пульт дистанционного управления бытовой техникой.

В чём польза проекта:

Решает извечную проблему потерянных или сломанных физических пультов. Позволяет управлять телевизорами, кондиционерами, ресиверами и другими устройствами прямо с экрана телефона в едином лаконичном интерфейсе.
